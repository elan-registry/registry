# Elan Registry v2.9.0 Release Notes

**Release Date:** September 30, 2025
**Type:** Minor Release - Documentation Restructure, Admin Automation & Email Enhancements

## 🚨 REQUIRED ACTIONS AFTER DEPLOYMENT

### Documentation System: Admin Page Permissions Update

**⚠️ All administrative pages need proper UserSpice permissions after deployment**

1. **Run Admin Permission Update Script** *(via FIX system)*
   - Navigate to `FIX/index.php` in admin interface
   - Execute script `14-Update-Admin-Page-Permissions.php`
   - Verify all admin pages are properly registered in UserSpice
   - Confirm successful permission assignment:
     - Administrator (role 1) has access to all admin docs
     - Editor (role 2) has access to all admin docs

2. **Verify Documentation System Access** *(via browser testing)*
   - Test public documentation access: `/docs/faq/index.php`
   - Test admin documentation access: `/docs/faq/admin/index.php`
   - Verify unified document viewer: `/docs/view.php?doc=CAR_TRANSFER_USER_GUIDE.md`

3. **Update Documentation References**
   - Update any bookmarks to use new documentation structure
   - Inform users of the new centralized documentation system

**🎯 Success Criteria:**
- ✅ FIX script completed without errors *(COMPLETED)*
- ⏳ All admin pages accessible to authorized users *(PENDING - requires testing)*
- ⏳ Documentation redirects working properly *(PENDING - requires testing)*
